# Images — Drop your files here

Place your photos in this `/public` folder with these exact names:

| Filename | Usage | Notes |
|---|---|---|
| `portrait.jpg` | About section — your headshot | The close-up red-wall portrait works great |
| `portrait-seated.jpg` | (Optional) alternate photo | Full-length seated version |
| `book-cover.jpg` | Author section — The Artist cover | The Rorschach cover image |

The site handles missing images gracefully — it shows a styled fallback — so you can deploy without them first and add photos later.

## Recommended sizes
- `portrait.jpg` — any portrait-ratio image, min 600×750px
- `book-cover.jpg` — 2:3 ratio, min 400×600px
